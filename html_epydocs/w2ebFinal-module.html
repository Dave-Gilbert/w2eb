FinalAddSections(opts, bl, sect_label_href_list)

source code 

Add the collected sections to the end of the book

Parameters:
  • opts - Dictionary of common CLI parameters. See StartupGetOptions()
  • bl - Beautiful Soup representation of an HTML web page.
  • sect_label_href_list - List of sections by label and href

Note:

The order of sect_label_href_list is important, and determined during book traversal. The tree of sections is flattened using a breadth first search. This means that all the references to any article appear in order after that article, even if those articles have their own subarticles.

FinalFixFirstRefLabel(bl, foot_dict_list)

source code 

Append 'a' for multi reference footnotes

Parameters:
  • bl - Beautiful Soup representation of an HTML web page.
  • foot_dict_list - List of footnotes represented by dictionaries

Note: When we generate a footnote for the first time we don't know if it will have multiple references, so we just guess that it won't. This function finds those foonotes, and appends a 'a' to their name.

FinalBackLinks(foot_dict)

source code 

Generate the backlinks for each footnote.

Parameters:
  • foot_dict - Dictionary storing details for notes and footnotes
Returns:
an html string with a list of links

Note: Most footnotes will be referred to once, but it is not unusual for a footnote to have multiple referents. We need to provide an appropriate backlink mechanism so we can return to any one of them.
