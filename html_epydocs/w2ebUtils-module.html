uHrefOk(tag_href, exclude_internal)

source code 

Verify that soup tag is an anchor with an href and other properties

Parameters:
  • tag_href - B Soup object representing a tag with an "href" attribute
  • exclude_internal - whether or not to allow internal references
Returns:
boolean, whether or not reference can be included

This function is used as part of a BeautifulSoup find_all() call to filter out tags that we will or wont try to make into footnotes.

When we search for tags to modify we don't want to change anything that refers to the current page. We have a separate search that also uses this filter to count all references for statistics purposes.
