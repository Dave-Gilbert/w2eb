TocShortLabel(heading_str)

source code 

Construct a simplified heading for the table of contents

Parameters:
  • heading_str - a string to be used for a table of contents heading
Returns:
a string suitable for a TOC entry

Note: Strings shorter than 40 characters or so are used as-is. We tinker with other strings by searching for more natural end points.

Verify curr and if okay add to our list of tags.

Parameters:

  • hdlist - a list of ids associated with the heading, for jump points
  • curr - the current candidate for TOC entry
  • head - the old TOC label
Returns:
a revised [hlist, hdlist] pair
